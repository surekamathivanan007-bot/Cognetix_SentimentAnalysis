# Social Media Sentiment Analysis

Project 9 - Advanced Stage
Internship: Cognetix Technology

## About
Analyzed public opinion from social media posts using NLP-based sentiment scoring with VADER to classify posts as Positive, Neutral or Negative.

## Dataset
Download from Kaggle: https://www.kaggle.com/datasets/kashishparmar02/social-media-sentiments-analysis-dataset
Place the file as sentimentdata.csv in the project folder.

## Libraries used
- pandas
- nltk
- matplotlib
- seaborn
- wordcloud

## How to run
pip install pandas nltk matplotlib seaborn wordcloud jupyter
jupyter notebook analysis.ipynb
